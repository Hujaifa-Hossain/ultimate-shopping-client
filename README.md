#### ULTIMATE SHOPPING

###### An awesome e-commerce website using

* ES6
* SCSS
* React JS
* Redux Toolkit
* RTK Query
* Mongoose
* Express JS
* Node JS
#### Demo

https://ultimate-shopping.netlify.app

#### Installation

* First download the file and then run these commands.
```bash
  cd ultimate-shopping-client
  npm install 
  npm start
```
